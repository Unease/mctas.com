A non-playback mod for version 1.8.9-1.18.1

#### Features

*   Tickratechanger
*   Tickrate 0/Pause server and client
*   Item duplication (Duping)
*   Savestates
*   Drop Manipulation
*   Dragon Manipuation
*   Spawn Manipulation
*   Ingame timer
*   And more!
